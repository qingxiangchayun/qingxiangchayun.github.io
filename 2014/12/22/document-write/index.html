document.write 介绍

markup is a string containing the text to be written to the document.

与DOM Ready相关 tips

  • 在DOM Ready之前执行document.write(), 会往页面上追加内容
  • 在DOM Ready之后执行document.write(), 会先清空页面内容，在写入新的内容；
    相当于 document.open() —> document.write() —> document.close()
